TSLStudio is a comprehensive library of TSL (Three.js Shading Language) node functions for procedural generation, signed distance fields, lighting, and advanced visual effects.

All examples use WebGPU rendering with Three.js r181+. These are real-time, GPU-accelerated shaders running entirely in your browser.
