   - Adapting existing code >> rewriting
   - Preserved original logic
   - Faster and more reliable

   - Easy to review

   - Writing docs during implementation
   - Context preserved
   - Higher quality
   - Production-ready from start

5. **Modular Architecture**
   - Separate utils and utils-matrix
   - Tree-shakeable exports
   - Easy to maintain
   - Room for growth

3. **Matrix transformations** need dedicated module
4. **Special channels** integrate beautifully with TSLFn
5. **Build system quality** enables confident iteration
